Publish scholarly journals using WordPress

== Description ==

**Still work in progress. Not yet ready for release.**

* Adds custom post type for articles and custom taxonomy for issues
* Automatically parses footnotes
* Automatically parses table of contents
* Automatically paginates articles
* Automatically tags articles
* Automatically links tags in articles
* Automatically links URLs in articles
* Paywall
* Font Toggle widget
* Issue list widget
